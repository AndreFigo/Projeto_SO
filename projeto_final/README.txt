To execute:
 -Compile with 'make';
 -Execute 'race' file to start the program;
 -Execute 'commands.sh' file to add cars and start the race (edit it if needed).

To signal SIGUSR1:
 -Race manager pid is printed to log.txt (ACCEPTING SIGUSR1 SIGNAL IN PID ...);
 -Use it with command kill to send the signal.
